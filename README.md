How to run the Park Ticketing Management System Project 

1. pen the PARK TICKETING MANAGEMENT SYSTEM folder and copy PTMSP folder

2.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

3. Open PHPMyAdmin (http://localhost/phpmyadmin)

4. Create a database with name ptmsp

5. Import ptmsp.sql file(given inside the package in SQL file folder)

6.Run the script http://localhost/PTMSP
